# -QT
Задание: Написать программу, которая умеет делать следующее: 
• Подключаться по TCP/IP к эмулятору прибора = серверное приложение
• Запрашивать версию прибора 
• Выводить информацию в GUI 
Предполагается решение задачи средствами Qt 5.15.2

Запрос версии осуществляется командой ‘v’, 0. Это значит, что команда состоит из двух байт, первый имеет значение ‘v’, второй 0. В ответ эмулятор отправит версию, которую нужно будет вывести куда-нибудь в GUI.
В данном контексте 0 – это число, а не символ.

Формат данных посылки обмена между устройствами ЯФИ и РС
	'S', Ll, Lh, Ksl, DATA[], Ksd
где:
 'S' – признак начала посылки
 Ll – младший байт длинны данных
 Lh – старший байт длинны данных
 Ksl – контрольная сумма длинны данных (Ll + Lh + Ksl = 0)
 DATA[] - массив данных длинной L
 Ksd – контрольная сумма данных (DATA[0] + … + DATA[L-1] + Ksd = 0)

Client.pro - главный pro-фаил
mainwindow.h - заголовочный фаил. Необходим для объявления классов и функций, используемых в программе
main.cpp - исходник. Содержит основное тело программы
mainwindow.cpp - исходник. Содержит описание создаваемых классов и методов
mainwindow.ui - форма. Представляет собой дерево виджетов формы в формате XML.
